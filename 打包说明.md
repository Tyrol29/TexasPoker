# 德州扑克 - EXE 打包说明

## 打包文件位置

打包完成后，可执行文件位于：
```
dist/德州扑克.exe
```

## 文件信息

- **文件名**: 德州扑克.exe
- **文件大小**: 约 8 MB
- **运行环境**: Windows 7/8/10/11 (64位)
- **依赖**: 无需安装 Python 或其他依赖

## 使用方法

### 1. 复制到目标电脑

将 `dist/德州扑克.exe` 文件复制到任何 Windows 电脑上，双击即可运行。

### 2. 命令行运行（可选）

```cmd
德州扑克.exe
```

## 重新打包

如果需要重新打包（例如修改代码后），执行：

```bash
python build_exe.py
```

## 注意事项

1. **杀毒软件**: 某些杀毒软件可能会误报 PyInstaller 打包的 exe 文件，这是正常现象，可以添加信任。

2. **文件路径**: 如果 exe 文件路径包含中文，在某些旧版 Windows 上可能会有问题，建议放在英文路径下。

3. **控制台窗口**: 这是一个控制台应用程序，运行时会显示命令行窗口。

## 自定义打包选项

编辑 `build_exe.py` 文件可以修改打包选项：

- `--onefile`: 打包成单个 exe 文件（推荐）
- `--onedir`: 打包成文件夹（启动更快，文件更小）
- `--name`: 修改输出文件名

## 常见问题

**Q: 为什么 exe 文件这么大？**
A: 因为包含了 Python 解释器和所有依赖库，这是正常现象。

**Q: 可以在 Mac/Linux 上运行吗？**
A: 不可以，这个 exe 只能在 Windows 上运行。需要在其他系统上运行时，需要在该系统上重新打包。

**Q: 如何减小文件大小？**
A: 可以使用 UPX 压缩工具，或者使用 `--onedir` 模式代替 `--onefile`。
